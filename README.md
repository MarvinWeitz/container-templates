# container-templates

Dev Container Templates published to GHCR at `ghcr.io/marvinweitz/container-templates`.

## Templates

| Template | Description |
|---|---|
| **`btp-generator`** | Scaffolding environment with `yo` and `@cbs-consulting/generator-btp`. Runs the generator interactively on first start. |

## Usage

In VS Code: **Command Palette → Dev Containers: Add Dev Container Configuration Files** and search for templates from `ghcr.io/marvinweitz/container-templates`.

Or reference directly in an empty project's `.devcontainer/devcontainer.json`:

```jsonc
// Copy from src/btp-generator/devcontainer.json
```
